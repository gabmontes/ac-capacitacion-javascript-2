# Clase 1

## Temario

- Nueva sintaxis:
  - Arrow functions y el alcance de `this`
  - Objetos literales: abreviaturas, propiedades computadas
  - Plantillas (template literals), plantillas etiquetadas
  - Desestructuración de arreglos, objetos, parámetros
  - Valores por defecto
  - Rest y spread en arrays (y en objetos)
  - `let` y `const`
- Extensiones:
  - `String` methods `includes`, `repeat`, `startsWith`, `endsWith`
  - `Array` methods `find`, `findIndex` (e `includes`)
  - `Object.assign`
  - `Promise`
  - `Map`, `Set`, `WeakMap`, `WeakSet`
- Otros cambios:
  - `Class`, `constructor`, `super`, métodos y propiedades
  - `Symbol`
  - Iteradores, iterables, for..of
  - Generadores
  - `Proxy`
  - Módulos (import, export)
